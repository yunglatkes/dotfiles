# Commands for Setting up Repo
## Initialization
```sh
# In bash.rc
alias dot='/usr/bin/git --git-dir=$HOME/.dotfiles/ --work-tree=$HOME'

echo ".dotfiles" >> .gitignore

git clone --bare git@github.com:yunglatkes/dotfiles.git $HOME/.dotfiles

# Delete conflicting files if present
dot checkout
```


## Fresh Installation
```
# Notice the --bare flag; this initializes the repository as a git bare
git init --bare $HOME/.dotfiles

# Add the following to .bashrc
alias dot="/usr/bin/git --git-dir=$HOME/.dotfiles --work-tree=$HOME"

# Don't display untracked files in git status
dot config --local status.showUntrackedFiles no
```
